## 三、账号定位

### 现在的问题
「校园墙」定位太模糊。用户看到你发新生指南、梗图、答疑帖……不知道你到底是谁、为什么值得关注。

### 建议定位

> **「在立信，有一个懂你的学姐叫小狐」**

核心标签：
- 🦊 **在读学姐**（有人设，不是冷冰冰的「墙」）
- 📢 **校园信息差终结者**（知道别人不知道的事）
- 🗣️ **敢问敢说**（争议话题、真实表达）
- ☕ **立信生活家**（咖啡、外卖、周边……日常但有用）

### 和竞品的差异

| | 立信小狐（你） | 上海立信-小狐（老号） | 小狮哥 |
|------|------|------|------|
| 人设 | 学姐+真实 | 工具型 | 学长+避坑 |
| 内容 | 综合 | 攻略为主 | 新生为主 |
| 差异点 | **争议话题 + 真实日常** | 信息量大 | 男生视角 |

你的护城河：**敢发别人不敢发的内容**（最雷的老师 265赞已证明）+ **微信私域 2500 好友的素材来源**。

## 四、内容策略

### 4.1 内容金字塔

```
        ┌─────────┐
        │  爆款级  │  每月1-2篇 — 争议/表达欲话题
        │  (目标100+)│  例：最雷的老师、最水的课、薪资爆料
        ├─────────┤
        │  互动级  │  每周1篇 — 征集/投票/评论区互动
        │  (目标20+) │  例：一张图证明你是立信人、吃瓜征集
        ├─────────┤
        │  攻略级  │  每周1-2篇 — 实用信息
        │  (目标10+) │  例：外卖攻略、App推荐、周边探店
        ├─────────┤
        │  日常级  │  每周2-3篇 — 人设日常
        │  (目标5+)  │  例：校园随手拍、咖啡探店、碎碎念
        └─────────┘
```

**目标发布频率：每周 4-6 篇**

### 4.2 各品类选题库

#### 🔥 爆款级（争议/表达欲）— 每月 1-2 篇

选题逻辑：让用户「有话说、不得不评」

- 「立信最水的课，你敢说出来吗？」

- 「如果重新选一次，你还会来立信吗？」
- 「立信毕业5年后，同学之间的差距有多大」
- 「立信vs上海其他双非，真实差距在哪」
- 「那些年立信老师讲过的离谱话」
- 「立信校招，哪些公司千万别去」
- 「在立信，一个月花多少钱才算正常」

#### 📢 互动级（征集/投票）— 每周 1 篇

选题逻辑：低门槛参与，评论区驱动流量

- 「一张图证明你在立信」（评论区带图）
- 「立信食堂红黑榜」（评论区征集）
- 「你的专业在立信属于哪个梯队？」（投票+讨论）
- 「立信猫猫图鉴」（征集+合集）

#### 📝 攻略级（实用信息）— 每周 1-2 篇

**不再做**：新生指南类（已饱和，21篇够用了）

**改做**：

#### 🌸 日常级（人设日常）— 每周 2-3 篇

## 五、标题方法论

### 「上海立信-小狐」为什么均赞高？标题是关键

| 你的标题 | 老号的标题 | 差距在哪 |
|------|------|------|
| 立信新生指南——快递篇 | 立信外卖怎么点 | 「指南」=说明书，「怎么点」=有悬念 |
| 立信新生指南——奖学金篇 | 张雪峰老师这样谈立信 | 蹭名人+好奇感 |
| 立信新生指南——选课篇 | 某通撤出交大园区始末 | 叙事感+吃瓜驱动 |

### 好标题公式

```
[悬念/冲突] + [立信相关] + [第一人称/情绪]

❌ 「立信新生指南——宿舍篇」
✅ 「在立信住了一年，我终于知道哪个宿舍最香」
✅ 「立信宿舍避雷：这三个楼千万别选」
✅ 「立信宿舍真实照片，你敢晒吗？」
```

## 七、增长策略

### 7.1 小红书内部

1. **评论区运营**：每条笔记发完后，自己先用小号发 2-3 条评论暖场（尤其「最雷的老师」这类需要用户留言的笔记）
2. **蹭热点**：高考出分、录取季、开学季、考试周——立信相关话题搜索量暴增时密集发布攻略级内容
3. **话题标签**：每条笔记带 `#上海立信会计金融学院` `#立信` `#上海大学生` `#松江大学城`
4. **合集功能**：把「新生指南」21篇整理成合集，方便新用户一键收藏

### 7.2 微信私域联动

1. **朋友圈征集 → 小红书发布**：在微信朋友圈征集照片/回答 → 整理成小红书笔记 → 在笔记中注明「via 微信好友投稿」→ 反向吸引小红书用户加微信
2. **小红书引流微信**：个人简介加入「投稿/进群→看瞬间」引导语，配合置顶笔记说明

### 7.3 开学季是最大增长窗口

每年 7-9 月是立信小红书搜索量巅峰（高考出分→填志愿→录取→开学），建议：
- **7月初**：批量准备 5-8 篇攻略级+爆款级内容
- **7月中-8月**：高频发布（每周5-7篇），争夺新生流量
- **9月开学**：转向日常级+互动级，留住新生关注
