### 运营约束

### 微信私域行业现状（2025-2026）
- **市场趋于饱和**：大多数高校已有多个"墙"类账号，新入局竞争激烈，业界出现"2026 不要再做表白墙"的声音
- **学校监管收紧**：越来越多高校出台管理规定，封禁非官方账号，合规化运营是趋势
- **从个人微信转向企业微信**：企微支持实名认证、群管理权限、消息互通，成为校园私域标配
- **小程序 + 私域双循环**：圈子+话题+帖子+商城四维一体，构建"内容-社交-交易"校园小生态
- **AI 赋能**：AI 自动回复、内容生成、情感分析预警等深度嵌入校园私域运营
- **分站/合伙人模式**：选拔学生 KOC 作为分站站长，收益共享，跨校裂变扩张
- **核心公式**：校园墙价值 = 覆盖该校学生比例 × 信任度 × 变现能力

## 可用 MCP 工具

### Playwright（浏览器自动化）
- 可以打开网页、点击、填表单、截图等，模拟真实浏览器操作
- **适用场景**：需要登录后才能访问的页面、JavaScript 渲染的动态页面、网页交互操作
- **注意**：操作需要时间，复杂任务请给足超时时间

### Firecrawl（网页抓取）
- 搜索、抓取网页内容，返回 Markdown 格式的结构化数据，支持批量处理和绕过反爬
- **适用场景**：搜索信息、批量抓取网页、获取 LLM 友好的内容
- **免费额度**：500 credits（一次性）

### Notion（笔记与知识管理）
- **已配置**：`.mcp.json`（gitignore，token 不泄漏），2026-05-07 验证连接成功
- 搜索、创建、更新、读取 Notion 页面和数据库，管理团队空间
- **适用场景**：将本地文档同步到 Notion、创建结构化笔记和任务看板、数据管理、个人知识库
- **注意**：重启 Claude Code 后需批准 MCP server 连接；若连接失败，检查 `NOTION_TOKEN` 是否过期
- **核心能力**：
  - `notion-search`：语义搜索 Notion 工作区内容（支持 AI 搜索和关键词搜索）
  - `notion-fetch`：获取页面/数据库/数据源的完整内容（返回 Notion-flavored Markdown）
  - `notion-create-pages`：创建新页面（支持指定父页面、数据库模板、图标/封面）
  - `notion-update-page`：更新页面属性和内容（支持属性修改、内容替换）
  - `notion-create-database`：创建数据库（SQL DDL 语法定义 Schema）
  - `notion-update-data-source`：更新数据源 Schema（添加/删除/重命名属性）
  - `notion-create-comment`：添加页面评论或回复讨论
  - `notion-get-comments`：获取页面评论和讨论
  - `notion-get-teams` / `notion-get-users`：查询团队空间和用户信息
  - `notion-move-pages`：移动页面到指定父页面
  - `notion-duplicate-page`：复制页面
- **Notion-flavored Markdown 要点**：
  - 表格使用 XML 格式（`<table>` / `<tr>` / `<td>`），支持表头行/列、列宽、颜色
  - 支持高级块：`<details>`（折叠）、`<callout>`（标注框）、`<columns>`（分栏）、`<page>`（子页面）
  - 支持 toggle heading：`# 标题 {toggle="true"}` 可折叠标题
  - 内联格式：`**粗体**`、`*斜体*`、`~~删除线~~`、`<span underline="true">下划线</span>`、`<span color="Color">彩色文字</span>`
  - 颜色列表：gray/brown/orange/yellow/green/blue/purple/pink/red（加 `_bg` 为背景色）
  - 待办：`- [ ]` / `- [x]`
  - 代码块：` ```language ``` `（支持 Mermaid 图表）
  - 转义字符：`\\ * ~ \` $ [ ] < > { } | ^`
